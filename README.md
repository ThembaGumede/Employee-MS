## Employee Management System

A modern, responsive employee management system built with React and Bootstrap. This application allows HR departments to manage employee records efficiently.

## Features

- Employee CRUD Operations: Create, Read, Update, and Delete employee records
- Responsive Design: Built with Bootstrap for mobile-friendly experience
- Modern UI: Clean and intuitive user interface
-  Functionality: Quickly find employees by name or department
- Department Management: Organize employees by departments

##  Tech Stack

- Frontend: React 18 + Vite
- Styling: Bootstrap 5 + Custom CSS
- Build Tool: Vite
- Package Manager: npm/yarn
